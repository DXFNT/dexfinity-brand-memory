# Dexfinity Emoji System

Kurátorovaný emoji set pre Dexfinity komunikáciu. Žiadny startup hype. Konzistentný význam naprieč kanálmi.

---

## 1. Zakázané emoji

| Emoji | Prečo |
|---|---|
| 🚀 | Startupovo, hypeovo, "rocket launch" cliché |
| 🔥 | Lacno-growthovo, "fire" hype |
| 🍻 | Neformálny "after party" feel, neprofesionálne |

Tieto emoji nesmú byť v žiadnom Dexfinity outpute. LinkedIn, email, deck, interná správa, kdekoľvek.

---

## 2. Preferovaná primárna paleta

Týchto 7 emoji nesie konkrétny brand význam.

| Emoji | Význam | Typické použitie |
|---|---|---|
| 💙 | Kultúra, tím, Dexfinity afekcia | Tímové updaty, "love" pre Dexfinity, kultúrne posty |
| 📘 | Know-how, expertíza, learning | Knowledge posts, frameworky, edukatívny obsah |
| 🌀 | AI, transformácia komplexity | AI témy, simplifikácia, transformation stories |
| 🌐 | Globálny rast, expanzia, multi-market | Cross-border témy, expansion case studies |
| 🔵 | Performance, precision, analytika | Performance reporty, data stories, metric showcases |
| 🔹 | Bullet points | Najčistejší bullet, dlhšie zoznamy v LinkedIn poste |
| 🧊 | Analytický pokoj, calm operator | Decision frameworky, strategické úvahy |

---

## 3. Sekundárna paleta

Týchto 5 emoji použiteľných pri špecifických momentoch.

| Emoji | Význam |
|---|---|
| ✨ | Príležitosti, momenty otvárania, "new opportunity" |
| ⭐ | Uznanie, milestones, klient achievements |
| 🛸 | Inovácie, future-forward, novel approach |
| 🧿 | Ochrana brandu, IP, security témy |
| 🪼 | Fluidné riešenia, adaptívne systémy |

---

## 4. Pravidlá použitia

### Per kanál

| Kanál | Emoji povolené | Pravidlá |
|---|---|---|
| Klientske audity | ❌ Nie | Executive tone bez emoji |
| Klientske prezentácie | ❌ Nie | Executive tone bez emoji |
| Klientske proposaly | ❌ Nie | Executive tone bez emoji |
| Emaily klientovi | ⚠️ Max 1 | Iba ak má presný význam (per paleta) |
| LinkedIn posty | ✅ Max 3 | Funkčné, nie dekoratívne, per paleta |
| Interné Basecamp | ✅ Áno | Per paleta |
| Interné Slack/WhatsApp | ✅ Áno | Per paleta, neformálnejšie |
| Event pages | ⚠️ 1 až 2 | Iba primary paleta |
| Partnership decks | ⚠️ Max 2 | Iba primary paleta |

### Per funkcia

- **Dekoratívny emoji = zakázaný.** Každý emoji musí niesť význam z palety.
- **Spam emoji = zakázaný.** Žiadne 5+ emoji na riadok.
- **Random emoji z mimo palety = vyhnúť sa.** Drž sa primary + secondary setu.
