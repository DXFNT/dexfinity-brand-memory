Every web page, HTML output, email template, presentation, or visual artifact must follow the Dexfinity design system exactly. Never use generic/default styles.

**Why:** User was frustrated that an audit page used wrong colors (#00D4AA teal), no logo, no circles, wrong fonts — completely off-brand. Dexfinity brand consistency is critical.

**How to apply:**
- Always invoke dex-design skill for any visual output
- Core colors: #05024E (navy), #0065F7 (blue), white, black
- Extended palette (from dexfinity.com): #16AB8B (teal) for CTA buttons and positive indicators, #ff7f00 (orange) for urgency/promo elements
- No reds, no traffic-light semaphores for severity — use navy/blue shades for severity, teal for CTA, orange for urgency
- Fonts: Questrial for headings, Poppins for body/UI
- Logo: use PNG URLs from dexfinity.com (Light variant on white, Dark variant on navy)
- Circles as decorative graphic elements (brand's "Lego brick")
- Tagline "Go Beyond" where appropriate
- Never modify the logo or create text-based logo alternatives like "Dexfinity Audit"
